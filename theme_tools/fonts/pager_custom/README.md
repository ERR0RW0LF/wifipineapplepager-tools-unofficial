## Character Set

As far as I have tested the following characters are supported:

```
ABCDEFGHIJKLMNOPQRSTUVWXYZ
abcdefghijklmnopqrstuvwxyz
0123456789!@#$%^&*()_+=[]{}
,;.:-_\/
```
